Initialize workspace for explicit 4th order Runge Kutta method.

This function will initialize the Runge Kutta 4th order integration to solve the differential equation d/dt y = f(y,t), where y is a vector and f is a arbitrary function for initial condition y(t0) = y0. The differential equation is solved using explicit Runge Kutta 4th order.

The callback function f must have type (*f)(matrix_complex_t *dest, matrix_complex_t *src, double t, void *args). The result must be stored in dest, the input vector is src, the time is t, and args is an arbitrary pointer given to the function.

Parameters
[in,out]selfexplicit Runge-Kutta 4th order object for real differential equation
[in]fcallback for rhs of differential equation
[in]dimdimension of differential equation (rows of vector y)
[in]argsarbitrary pointer that will be passed to f
Returns
ret 0 if successfull, otherwise LIBHADES_ERROR_OOM

Initialize symplectic Runge Kutta 4th order integration.

This function will initialize the Runge Kutta 4th order integration to solve the the differential equation d/dt y = f(y,t), where y is a vector and f is a arbitrary function. The differential equation is solved using implicit, symplectic Runge Kutta 4th order.

The function f must be of type void f(matrix_t *ft, matrix_t *y, double t, void *args). The result of f will be stored in ft, y and t are the arguments, args is a pointer to arbitrary data.

Parameters
self[out] symplectic Runge Kutta object
f[in] callback of function f
args[in] arbitrary data that is passed to f and Jf
