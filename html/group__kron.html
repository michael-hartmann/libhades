Calculate Kronecker product of real matrices A and B.

Matrix A has dimension Am,An, matrix B has dimension Bm,Bn.

If C is not NULL, the Kronecker product will be stored in C. C must have dimension Am+Bm,An+Bn. If C is NULL, memory for the matrix will be allocated and the matrix will be returned. You have to free the memory for the returned matrix yourself.

Parameters
[in]Areal matrix
[in]Breal matrix
[in,out]CKronecker product
Return values
CKronecker product of A and B, NULL if no memory could be allocated
