int matrix_lu_decomposition ( matrix_t *  A,
int  ipiv[] 
)

Compute LU decomposition of real matrix A.

See dgetrf.

The factorization has the form A = P * L * U where P is a permutation matrix, L is lower triangular with unit diagonal elements (lower trapezoidal if rows > columns), and U is upper triangular (upper trapezoidal if m < n).
